## 0. 한 줄 정의
평소 **옷 착용감**(부위별 느낌)을 입력 → **8체형 유형**으로 판정 → **공유용 결과 카드**(도트 캐릭터 + 핏 가이드)를 보여준다.
- 분류 근거 = **사이즈코리아 8체형**(공식) / 표현 = **우리 브랜드 네이밍·캐릭터**
- 별개 엔진: **브랜드별 사이즈 번역표**(같은 결과 페이지에 공존, 로직은 독립)

## 2. 진단 로직 (입력 → 유형)

### 입력
- **성별** (여/남) — 캐릭터 머리만 분기(MVP)
- **부위별 느낌**: 어깨·가슴·소매·배·총장(상의) / 허리·힙·허벅지·기장(하의) → 각 `큼 / 여유 / 딱맞음 / 끼임` (총장·기장은 `긺 / 짧음`)

### 신호 해석
| 느낌 | 의미 | 신호 |
| --- | --- | --- |
| 끼임 | 그 부위가 옷보다 큼 | **볼륨 +** |
| 여유·큼 | 옷이 남음 | **슬림 +** |
| 딱맞음 | 표준 | 중립 |
| (총장) 긺 / 짧음 | 기장 | 컴팩트(C) / 롱(L) |

### 8유형 판정 휴리스틱 (부위 신호 → archetype)
| 유형 | 판정 신호 |
| --- | --- |
| STR 사각 | 어깨≈가슴≈힙, 허리 여유 적음 (전체 비슷·굴곡 적음) |
| TRI 삼각 | 힙·허벅지 **끼임**, 어깨·가슴 여유 |
| INV 역삼각 | 어깨·가슴 **끼임**, 힙 여유 |
| HRG 모래시계 | 어깨·힙 끼임 + 허리 **많이 여유**(잘록) |
| BAL 표준 | 대부분 **딱맞음**(균형) |
| DIA 마름모 | 허리·배 **끼임**, 어깨·다리 여유 |
| RND 둥근 | 전체적으로 **끼임**(볼륨) |
| TUB 튜브 | 전체적으로 **여유·큼**(가늘고 굴곡 없음) |

> 동점·전부 딱맞음 → 저신뢰 → "옷 더 추가" 유도. 정밀 분류(둘레 실측·사진)는 v2.
> 프레임(롱 L / 컴팩트 C)은 **2차 태그**(총장·기장 기반), 카드에 `· 롱` 식으로 부가 가능.

## 3. 디자인 토큰

### 색
```css
/* 페이지(라이트) */
--bg:#ffffff; --bg2:#f7f8fa; --ink:#191a1c; --sub:#71717a; --sub2:#a6a7ae;
--line:#eeeff1; --line2:#e3e4e8; --soft:#f2f3f5; --ac:#2563eb;
--ok:#2f9e6f;          /* 정확도/성공 신호 전용 */

/* 결과 카드(다크) */
card-bg: radial-gradient(ellipse 70% 34% at 50% 11%, {POINT}2b, transparent 60%),
         linear-gradient(160deg,#191a1f,#0f1014);
/* {POINT} = 유형별 포인트색. 카드 위 글로우로만 색을 씀(나머지 흰 텍스트) */
```
**원칙:** 카드는 블랙 + **유형 포인트색 1개**만(실루엣·코드·궁합·아이콘에). 무채색 베이스.

### 카드 구조 (위→아래)
```
┌ ctop: fitting 로고 ─ [🔖 저장][🔗 공유] 아이콘
│ 도트 캐릭터 (성별 머리 + 실루엣 바디)
│ CODE (예: HRG)             ← 포인트색
│ 이름 (예: 엘레강스 X라인)
│ MY FIT      : 프로필 2줄
│ FIT INSIGHT : 한 줄 행동 팁
│ 잘맞 FIT / 피할 FIT : 쉬운 말 2개씩 (구체 아이템만)
│ (flex grow)
└ 🩷 환상의 궁합 · {match}    ← 맨 아래 고정
```
- **구조 고정:** 카드 `min-height` + 캐릭터 영역 높이 고정(184px) + 섹션별 고정 높이 → 유형 바꿔도 박스 안 흔들림.

### 도트 실루엣 (clip-path)
바디 = `width:120px; height:150px;` + 8px 도트 그리드(repeating-linear-gradient) + 유형 `clip-path` (값은 JSON `silhouetteClipPaths`).

### 캐릭터 (성별 = 머리)
```
얼굴 skin: #ecd9c9 (+미세 도트 그리드) · 눈: #2a2b30
여성: 단발(턱선까지) · 갈색 #9a7350  (longhair + cap)
남성: 짧은 머리(위만) · 검정 #2a2b30  (cap)
```

### 타이포
- 폰트: **Pretendard** (fallback: Apple SD Gothic Neo)
- 이름 32px/800 · 코드 12.5px/800(자간 .22em) · 본문 14px

## 5. 통합 노트 (받는 쪽 주의)
1. **8유형 = 식별/스타일 가이드**, **사이즈 번역(브랜드별 사이즈) = 별개 엔진** — 한 결과 페이지에 공존, 데이터 소스 다름.
2. **성별**은 현재 *머리만* 다름. 추천 분기(여성 원피스·치마, 남성 제외)는 v2 확장 포인트.
3. **프레임(롱/컴팩트)** 은 2차 태그 — 필요 시 카드 이름 옆/하단에 노출.
4. **포인트색·실루엣·문구**는 전부 JSON 데이터 주도 → 카드 컴포넌트는 데이터만 바꿔 재사용.
5. **킬 메트릭**(별개): 추천 사이즈 동의율 ≥65% — 카드 정확도가 아니라 *사이즈 번역* 검증 지표.

## 6. 빠른 재현 가이드
1. `진단카드_8유형.json` 로드 → 유형 객체 배열.
2. 카드 컴포넌트: ctop / 캐릭터(머리+clip-path 바디) / code / name / MY FIT(profile 2줄) / FIT INSIGHT / 잘맞·피할 / 궁합.
3. 카드 배경 = 다크 + 유형 `point` 글로우. 실루엣·코드·궁합·아이콘에만 포인트색.
4. 성별 토글 = 캐릭터 머리(여 단발 갈색 / 남 검정) 스위치.
5. (선택) 결과 페이지엔 우측에 브랜드별 사이즈표 + 정확도 검증 + 전문가 전환 CTA.
