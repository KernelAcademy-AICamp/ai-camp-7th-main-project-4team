### 1. 한 줄 요약
스타일에 자신 없는 사람이, 중요한 순간(결혼식·여행)에 **사이즈+핏으로 체형을 진단**받고 → **검증된 쇼퍼**가 옷을 골라줘 **"한 번에 잘 입게"** 하는 서비스.

### 2. 문제 정의
스타일에 자신 없는 사람이 — 특히 잘 입어야 하는 순간에 — **"내 체형에 뭐가 맞는지"를 몰라** 확신이 안 선다. 지금은 비슷한 체형 후기를 일일이 뒤지다 무난한 걸 사고 → **실패 → 안 입는 옷**으로 이어진다.

### 3. 타깃 페르소나 · JTBD
- **김지우 · 28세 · 키 158 · 온라인 쇼핑 주 1회+** — 표준 사이즈가 안 맞아 매번 막히고, 3주 뒤 친구 결혼식 하객룩이 막막. "나한테 맞는 걸 누가 딱 골라줬으면."
- **JTBD:** *When* 잘 입어야 하는 순간에, *I want to* 내 체형에 맞는 옷을 확신 갖고 고르고, *So I can* 실패·반품 없이 "이번엔 잘 입었다"가 된다.

### 4. North Star Metric
중요한 순간마다 '체형진단 → 검증된 쇼퍼 매칭'을 거쳐 **"이번엔 잘 입었다"고 만족한 [매칭 완료 건수]**.
(가입자·진단 수 같은 허영 지표 ❌)

### 5. Aha Moment
가입 후 **3분 안에, 사이즈+핏 입력만으로 '내 체형 캐릭터' 결과 카드**를 받으며 *"오, 나 이런 체형이었구나 / 정확한데?"* 를 느끼는 순간. (줄자·전문가 없이 = 진입장벽 0)

### 6. MVP 기능 (핵심 2개 + IA)
**IA:** 홈 / 체형진단 / 쇼퍼찾기 / 마이 (강제 플로우 아닌 탭 구조)

**① 체형진단 (우리 특화)**
- 입력: 키·몸무게(선택) · **평소 상의/하의 사이즈** · **핏 진단 체크리스트**(어깨·가슴·허벅지·힙 끼임 / 기장 / 핏 뜸) · 손목 굵기 · 살 찌는 부위 · **전신사진(선택)**
- 출력: **6가지 체형 캐릭터** 중 하나(🔺역삼각 카리스마·🍐글래머 페어·📏시크 스트레이트·🌊러블리 웨이브·🦒내추럴 프레임·⚖️골든 밸런스) + 잘 맞는 핏 / 피할 핏 / 사이즈 팁 결과 카드
- *MVP: 자동 점수 로직 또는 사람이 수동 판정해도 무방*

**② 쇼퍼 매칭 + 검증**
- 체형 프로필을 기반으로 **매칭도 순 쇼퍼 추천** → 쇼퍼 상세는 **결과물·후기로 검증**(얼굴·팔로워 ❌) → 단발 패키지 결제(5~15만) → 쇼퍼가 추천 룩·구매처 전달
- *MVP: 쇼퍼 2~3명, 매칭·전달은 수동(위저드 오브 오즈)*

### 7. 안 할 것 (Out of Scope)
퍼스널컬러 진단 · AI 자동 추천 · 구독 모델 · 자체 커머스(재고) · 회원 시스템 풀빌드 · 쇼퍼 셀프 온보딩 → **전부 v2 이후**

### 8. 성공 기준
- 체형진단 완료율 (Aha 도달) **≥ 60%**
- 진단 → 매칭 결제 전환율 (가설)
- **"잘 입었다" 만족률 ≥ 70%**
- 다음 이벤트 **재호출 의향** 확보
