## 0. 원칙 (왜 이렇게)

- **색은 아껴 쓴다.** 화면은 흑·백·회색으로, 색은 *진짜 중요한 데*에만.
- **액션 = 검정, 강조 = 블루(아주 조금).** 버튼·선택은 검정으로 묵직하게, 블루는 "AI가 만든 값"(매칭도·추천 사이즈)에만 콕.
- **성공 신호 그린은 분리.** 킬메트릭("사이즈 맞음") 같은 기능 신호 전용 — 브랜드 포인트와 안 섞음.

## 1. 색 토큰

```css
:root{
  /* 무채색 베이스 */
  --bg:#ffffff;      /* 기본 배경 */
  --bg2:#f7f8fa;     /* 섹션·카드 보조 배경 (쿨 그레이) */
  --ink:#191a1c;     /* 본문·버튼·선택(검정) */
  --sub:#71717a;     /* 보조 텍스트 */
  --sub2:#a6a7ae;    /* 더 옅은 텍스트·플레이스홀더 */
  --line:#eeeff1;    /* 얇은 구분선 */
  --line2:#e3e4e8;   /* 입력/카드 테두리 */
  --soft:#f2f3f5;    /* 선택 보조 배경·태그칩 */

  /* 포인트 (아주 작게) */
  --ac:#2563eb;      /* 차분한 블루 — 매칭도·추천 사이즈·토글·포커스 */
  --ac-d:#1d4ed8;    /* 블루 hover */
  --ac-soft:#e8eefc; /* 블루 옅은 배경 (필수 뱃지·노트) */

  /* 기능색 (분리) */
  --ok:#2f9e6f;      /* 성공·"맞음" 신호 전용 (브랜드 아님) */
}
```

### 색 사용 규칙 (어디에 무엇)

| 요소 | 색 |
| --- | --- |
| 주요 버튼 (CTA) | **검정** `--ink` (hover `#000`) |
| 선택된 탭·칩·옵션 | **검정** `--ink` 채움 |
| 조건 배너 | **검정** 바 |
| 로고·제목·본문 | `--ink` / `--sub` |
| **매칭도 뱃지** | **블루** `--ac` |
| **브랜드별 추천 사이즈 값** | **블루** `--ac` |
| 토글 ON · 입력 포커스 | **블루** `--ac` |
| "AI 매칭 →" 링크 · 필수 뱃지 | 블루 `--ac` / `--ac-soft` |
| 사진 자리(커버·포트폴리오) | 회색조 그라데이션 (`#1c1d22`~`#9697a3`) |
| "맞음/성공" 신호 | `--ok` 그린 (여기에만) |

> 한 화면에서 블루가 보이는 지점은 **2~4곳을 넘지 않게.**

## 2. 형태 토큰

```css
/* 라운드 */
--r-btn:12px;  --r-input:12px;  --r-chip:30px;
--r-card:18px; --r-modal:24px;

/* 테두리 */
border:1.5px solid var(--line2);   /* 입력·옵션 */
border:1px  solid var(--line);     /* 카드·구분선 */

/* 그림자 (옅게) */
card-hover: 0 14px 34px rgba(20,22,28,.09);
drawer:     -22px 0 60px rgba(0,0,0,.18);
```

- **간격:** 섹션 라벨(`.feat`) 위 22px / 입력 라벨 위 24px / 카드 그리드 gap 20px.
- **컨테이너 폭:** 1080px, 좌우 패딩 28px.

---

## 3. 타이포

- 폰트: **Pretendard** (없으면 Apple SD Gothic Neo → sans-serif)
- 자간: `letter-spacing:-.02em` (제목은 -.045em)
- 위계: H1 32px/800 · H2 28px/800 · 본문 15~16px · 보조 13~14px · 라벨 12~13px/700

---

## 4. 컴포넌트 빠른 참조

| 컴포넌트 | 기본 | 선택/활성 |
| --- | --- | --- |
| 탭 | 회색 텍스트 | 검정 텍스트 + 검정 밑줄 |
| 칩(필터) | 흰 배경·회색 | 검정 채움·흰 글자 |
| 버튼(주요) | 검정 채움 | hover `#000` / 비활성 `--line2` |
| 버튼(보조) | 흰 배경·테두리 | hover `--soft` |
| 옵션 카드 | 테두리 `--line2` | 검정 테두리 + `--soft` 배경 |
| 토글 | 회색 | **블루** |
| 매칭도 뱃지 | — | **블루** |

---

### 적용 메모
- 세 프로토타입 모두 위 토큰으로 통일 완료. 색을 바꾸려면 각 파일 `:root`의 `--ac`(블루)·`--ink`(검정) **두 줄만** 고치면 전체 반영됨.
- 카드 커버는 실제 서비스에선 **전문가 결과물 사진**이 들어갈 자리 (지금은 회색조 placeholder).
