## 핵심만 빠르게
- **문제**: 무신사 같은 몰은 자기 안 + 구매이력으로만 사이즈를 추천해, **새 브랜드로 넘어가는 순간** 아무도 번역해주지 않는다.
- **해결**: "평소 잘 맞는 옷"의 착용감을 거꾸로 풀어 체형을 추정하고, 다른 브랜드에서 맞을 사이즈로 번역.
- **원칙**: 사이즈 계산은 **규칙 기반**(검증 가능), 자연어 서술만 LLM. 진단은 **옷 종류 단위**(기반=상의+하의). 성패는 **킬 메트릭 ≥65%** 하나로 판단.
- **디자인**: 오프화이트 + 딥그린 1포인트 + 잉크블랙. 결과 카드만 예외(다크+8유형색).
