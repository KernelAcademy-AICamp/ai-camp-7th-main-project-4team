## 발표 운영 팁

- **GIF가 기본, 라이브는 백업** — 발표 PC에서 `npm run serve` 후 `http://localhost:8000` 열어두면 질의응답 때 라이브 시연 가능
- GIF 프레임 전환 1.4~1.7초 · 마지막 프레임 2.4초 홀드 — 09번 장 멘트(1:15)와 맞춰짐
- 캡처 데모 데이터: 남성 172cm/68kg · 상의 슬림/하의 와이드 취향 · 착용경험 2벌(정확도 '높음' 상태)
- 재캡처가 필요하면(화면 바뀜) 팀장에게 — 캡처 스크립트로 10장 일괄 재생성 가능
