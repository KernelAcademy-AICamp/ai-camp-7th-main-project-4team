- **오류·빈 상태(G.7)** — **헤더/푸터 없는 공용 오류 페이지**(사용자·전문가·관리자). ?e=유형별 카피(notfound/expired/network) + 컨텍스트 감지(?app= 또는 referrer→admin/pro/user)로 '홈으로'를 해당 앱 홈으로, '이전 페이지로'=history.back. Vercel 404.html 컨벤션 + serve.js 라우팅(noindex).
- **데모 초기화 — 비배포** — sessionStorage 리셋 유틸. gen-app 미배포.
- **서비스 소개·작동원리** — 소개→diag-basic 전환 CTA 있음.
- **관리자 · B2B API(3.D) — v3** — **MVP 범위 밖 → 네비 숨김·미배포.** API 고객·플랜·키·기술지원 콘솔(샘플). 실 게이트웨이·과금은 v3.
- **관리자 · 브랜드 관리·노출순서(3.A.1)** — 브랜드별 커버리지·앵커·다양성·갭. + **진단 노출 순서 편집**(brand 테이블·db/04): 기본=실착 접근성 순(앵커 상위), admin이 순서 숫자·노출 토글 저장→다음 진단 반영. 실측표/수집 진입점.
- **관리자 · 사업현황(3.C.1) — v2** — **MVP 범위 밖 → 네비 숨김·미배포.** 운영 대시보드(퍼널·KPI 샘플)·금전(GMV·정산)은 v2. 진단·정확도는 admin-diagnostics로, 수요는 admin-leads로 이관됨.
- **관리자 · 이미지 붙여넣기 수집(3.A.1)** — 사이즈표 이미지→CSV 3방법: 수동전사·OCR(베타)·AI프롬프트 브릿지. CSV→build-sizespec.py→garments.json→import(대량 시드). 브랜드 목록은 garment 테이블(DB) 읽기.
- **관리자 · 진단 응답·정확도(3.C.2)** — 킬 메트릭(정확도) 집계. **실 DB(Supabase feedback+diagnosis 임베드·RLS admin)** 연결 + 로컬로그/샘플 소스 전환. **정확도 추세**(날짜별·엔진버전별 맞음율) 추가 → 엔진이 강해지는지 확인. 정확도×신뢰도 캘리브레이션·8유형/앵커/페인 분해·응답로그.
- **관리자 · 엔진 시뮬레이터·튜닝(3.A.3)** — 몸→사이즈 파이프라인을 실엔진(BodyModel·FitEngine)으로 재현. 신체추정→추천→핏스코어 분해→사이즈 래더+목표여유 what-if→파라미터. 실측표는 garment 테이블(DB) 읽기로 전환(로컬 폴백=garments.json).
- **관리자 · 브랜드·제품 실측표 CRUD(3.A.1)** — garment 테이블(DB)을 사이즈표(사이즈×부위 cm)로 제품별 표시 + **직접 CRUD**(사이즈 라벨·cm 인라인 편집·행 추가/삭제·저장). 저장=현재 뷰 insert 후 기존 id 삭제(교체). 변경 시 rev↑ → **진단(/api/diagnose)에 즉시 반영**. 대량 수집은 CSV→build-sizespec.py→import 유지.
- **관리자 · 엔진 강화(3.C.4)** — 진단 로그(diagnosis.input painFlags·openNote + feedback)를 **엔진 강화 작업목록**으로(docs/6 §1.1 성장구조·자동학습 아님). ① 미표기∩페인(브랜드×병목부위 끼임)→A축 역산 실측보정 ② 새 페인(자유서술)→PAIN_FLAGS 승격 ③ 확신 틀림→규칙 튜닝 ④ 수요 대비 데이터 부족→수집. 각 행이 실측표·수집으로 연결. RLS admin.
- **관리자 · 전문가 수요(3.C.3)** — 스타일리스트 매칭 수요(lead 테이블·db/02·RLS admin) 집계. KPI(총 수요·오픈알림·견적요청·이메일 확보) + **진단→수요 전환**(lead.session_id↔diagnosis.session_id 조인) + 서비스·상황별 + 수요 목록(이메일·유입 진단후/직접). 페이크도어 측정축.
- **관리자 로그인(구글 OAuth)** — Supabase Auth Google OAuth 실연결 + is_admin(admin_user 멤버십) 확인 → 세션 브릿지(fitting.admin)→admin.html. supabase-js 미로드 시 목업 폴백.
