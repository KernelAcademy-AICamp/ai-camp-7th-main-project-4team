## 비고
- **데모 초기화** — sessionStorage 리셋 유틸.
- **서비스 소개·작동원리** — 소개→diag-basic 전환 CTA 있음.
- **결과 공유 카드** — OG 공유 카드(소유자 아티팩트). card.js가 초대링크 index.html?from=CODE 생성 → 초대받은 사람은 홈으로 전환. 단 card 페이지 자체엔 뷰어용 '나도 진단' CTA 버튼은 없음.
- **기본 정보 입력** — 성별·키·몸무게. 성별 선택 시 평균값 프리필.
- **착용경험 보정 입력** — 앵커 브랜드·사이즈 완전 데이터화(#20). 엔진 역산 실연결은 상의(TOP) 기준.
- **진단 로딩** — 역산+매칭 계산 대기 → result로 이동.
- **홈·허브(탭 홈/쇼퍼찾기/마이)** — 단일 페이지에 탭 3섹션(#home/#shop/#my). 쇼퍼찾기=페이크도어.
- **견적 요청** — #17에서 개편. 수요자 인라인 견적(index #shop)과 표면 관계 정리 필요.
- **쇼퍼 지원(공급자)** — 공급자 진입.
- **진단 결과** — 상의만 실계산, 파생 카테고리는 선호핏 폴백/게이트.
