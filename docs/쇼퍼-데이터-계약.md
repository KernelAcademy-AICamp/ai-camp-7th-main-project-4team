# 쇼퍼 데이터 계약 — 쇼퍼찾기 ↔ 쇼퍼 지원 포털

> 사용자앱 **쇼퍼찾기**(`web/js/index.js`의 `EX`)와 **쇼퍼 지원 포털**(`web/js/pro*.js`의 `fitting.pro.profile`)의
> 쇼퍼 데이터를 통일하기 위한 합의. 담당: 디자이너(쇼퍼찾기) · 개발자(포털). **이 문서가 정본.**

## 배경
포털에서 쇼퍼가 입력한 프로필이 쇼퍼찾기의 카드·상세·필터로 그대로 흘러가야 한다.
현재 두 쪽의 필드명·코드값이 어긋나 있어 아래처럼 맞춘다.

---

## ✅ 확정 사항

### 1. 서비스 코드 = `image` 로 통일
- 3종: `online`(온라인 스타일링) · `shopping`(동행 쇼핑) · `image`(이미지 컨설팅)
- 포털의 `imaging` → **`image`** 로 변경.

### 2. 서비스는 다중(`services[]`) + 서비스별 속성을 쇼퍼에게 입력받음
쇼퍼가 여러 서비스를 등록할 수 있고, 서비스마다 아래 값을 받는다.

| 서비스 | 제공 방식 | 활동 지역 | 예상 기간 |
|---|---|---|---|
| 온라인 스타일링 | **비대면**(타입 고정) | — | 쇼퍼 입력 |
| 동행 쇼핑 | **대면**(타입 고정) | 쇼퍼 **선택** | 쇼퍼 입력 |
| 이미지 컨설팅 | **대면**(타입 고정) | — | 쇼퍼 입력 |

- 구조(안): `services: [{ type, price, mode, dur, regions? }]`
  - `mode` = 서비스 타입에서 고정(온라인→비대면, 동행·이미지→대면)
  - `regions` = 동행 쇼핑만
  - `dur`(예상 기간) = 쇼퍼 입력 — **입력 형식은 미확정**(아래 ⏳ 참고)

### 5. 소개(bio)
- 포털이 **경력 소개를 빼고 소개(bio) 하나만** 남김 → 그대로 쇼퍼찾기 `bio` 로 사용.

### 6. 포트폴리오
- 포털 구조로 통일: **사진 + 착용 모델 키·몸무게(cm·kg)**.
- 현재 쇼퍼찾기 상세 갤러리엔 cm·kg 표기가 빠져 있어 **추가 필요**.

---

## ⏳ 미확정 — 레퍼런스 리서치 후 결정
다른 서비스(숨고·크몽 등)가 어떻게 하는지 더 찾아본 뒤 확정한다.

- **상황(`occ`)** — 소개팅·면접·발표·결혼식 하객·여행·일상 코디. 쇼퍼찾기 **필터의 핵심**인데 포털 입력이 없음.
  → 포털에 "주력 상황" 선택을 넣을지 / 전문분야 태그로 대체할지.
- **전문분야(`specialties`/`tags`)** — 필드명·태그 풀 통일 방식.
- **예상 기간(`dur`)** — 입력 형식(자유 입력 / 구간 선택 등).

---

## 필드 매핑 (포털 → 쇼퍼찾기)

| 포털 `fitting.pro.profile` | 쇼퍼찾기 `EX` | 상태 |
|---|---|---|
| `name` | `nm` | ✅ |
| `services[] {type, price}` | `services[] {type, price, mode, dur}` | ✅ 구조 통일(속성 추가) |
| `specialties[]` | `tags[]` | ⏳ 필드명·태그 풀 |
| `bio` | `bio` | ✅ (경력 소개 제거) |
| `portfolio[] {src, height, weight}` | 상세 갤러리(사진 + cm·kg) | ✅ cm·kg 표기 추가 |
| `regions[]` | 동행 쇼핑 서비스 속성으로 | ✅ 서비스별 속성 |
| (없음) | `occ[]` | ⏳ 미확정 |
| — | `rating` · `review` · `match` | 파생값(입력 아님) |

---

## 반영 담당
- **포털(개발자)**: `imaging`→`image` · `services`에 `mode`/`dur`/`regions` 입력 · 경력 소개 제거
- **쇼퍼찾기(디자이너)**: `EX`를 `services[]`로 전환 · 목록·필터·상세·요청 화면 개편 · 포트폴리오 cm·kg 표기

> 참고 · 화면 변화 예시: 다중 서비스 도입 시 목록/필터/상세/요청 4화면이 어떻게 바뀌는지는 별도 목업으로 공유.
