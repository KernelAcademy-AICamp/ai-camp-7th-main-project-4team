# 핏팅 QA 트래커 (노션 DB)

> 사용법: 아래 표를 복사 → 노션 붙여넣기 → 표 블록 선택 → **"데이터베이스로 전환(Turn into database)"**
> 그럼 필터·정렬되는 DB가 됩니다. 아래 **컬럼(속성) 설계**대로 각 컬럼 타입을 선택(Select)으로 바꿔주세요.

## 컬럼(속성) 설계

| 컬럼 | 타입 | 값(선택지) | 무엇 |
|---|---|---|---|
| **제목** | 제목 | (자유) | 이슈 한 줄 요약 |
| **화면** | 선택 | index · about · diag-basic · diag-fit · diag-loading · result · card · pro · pro-signup · pro-quote | 어느 화면에서 |
| **트랙** | 선택 | 🙋 사용자 · 🛍️ 쇼퍼 · 🔗 공통 | 누구 담당 영역 |
| **종류** | 선택 | 🎨 디자인 · ✨ 매끄러움(UX) · 🔀 흐름(IA) · ⚙️ 기능 · ✍️ 카피 | 무슨 이슈냐 |
| **심각도** | 선택 | 🔴 높음 · 🟡 보통 · ⚪ 낮음 | 얼마나 급한가 |
| **상태** | 선택 | 📥 열림 · 🔧 진행중 · ✅ 완료 | 지금 어디까지 |
| **담당** | 사람 | (팀원) | 누가 고칠지 |
| **비고** | 텍스트 | (자유) | 재현 방법·스크린샷·의견 |

### 종류(제일 헷갈리는 컬럼) 뜻
- **🎨 디자인** — 보이는 것. 색·크기·간격·정렬·폰트가 이상/불일치
- **✨ 매끄러움(UX)** — 동작이 어색·불편. 전환이 뚝 끊기거나, 눌러도 반응 없거나, 헷갈림
- **🔀 흐름(IA)** — 화면 이동이 플로우(기준)랑 다름. 다음 화면이 이상하거나 빠짐
- **⚙️ 기능** — 아예 안 됨. 버튼 눌러도 작동 X, 값 안 넘어감
- **✍️ 카피** — 문구·말투. 요체 안 맞음, 버튼 이름 어색
