# 사이즈 수집 템플릿 사용법

파일: `사이즈수집-템플릿.csv`

## 열(column) 설명

| 열 | 설명 | 예시 |
|---|---|---|
| brand | 브랜드명 | 무신사스탠다드, 유니클로, 자라 |
| fit | 핏 (슬림핏 / 루즈핏 / 오버핏) | 루즈핏 |
| product | 제품명 (색상 구분 불필요) | 릴렉스 핏 크루 넥 티셔츠 |
| size | 사이즈 라벨 | S, M, L, XL, 2XL ... |
| 총장 | 옷 길이 (cm) | 72 |
| 어깨너비 | 어깨 (cm) | 53 |
| 가슴단면 | 가슴 평면 단면 (cm) | 58 |
| 소매길이 | 소매 (cm) | 25 |
| 비고 | 메모 (선택) | 돌먼슬리브 등 |

## 입력 규칙 (중요)

1. **품목 통일** — 핏 비교를 하려면 같은 품목끼리만. 예: 전부 "반소매 티셔츠". 상의·하의 섞으면 평균이 무의미.
2. **한 행 = 한 제품의 한 사이즈** — 같은 제품의 S/M/L은 각각 한 행.
3. **단위 = cm, 숫자만** — "53cm" ❌ → "53" ⭕
4. **빈칸은 비워두기** — 측정 항목이 없으면(예: 돌먼슬리브 어깨너비) 0 대신 공란.
5. **핏 표기 통일** — 슬림핏 / 루즈핏 / 오버핏 세 가지로. (레귤러는 별도로 추가 가능)

## 제출 방법

아래 중 편한 것 하나:
- **A. 프로젝트 폴더에 저장** — 구글시트에서 `파일 → 다운로드 → CSV` 받아 이 폴더(`research/`)에 넣기 (권장)
- **B. 채팅에 붙여넣기** — 표를 복사해서 대화창에 붙이기 (양 적을 때)
- **C. 웹에 게시(CSV)** — 구글시트 `파일 → 공유 → 웹에 게시 → CSV` 링크 전달
- ⚠️ 일반 구글시트 `/edit` 공유 링크는 로그인이 걸려 직접 열 수 없음 → A/B/C 중 하나로.

## 제출하면 계산해 드리는 것

- brand × fit × size 별 **평균** + 표본 수(n) + **표준편차**(편차 큰 브랜드 식별)
- **핏 비교표** (같은 사이즈에서 핏별 치수 차이)
- **이상치/오타 점검** (예: 가슴단면이 비정상적으로 작으면 플래그)
- 결과를 `research/`에 마크다운/CSV로 정리·커밋
