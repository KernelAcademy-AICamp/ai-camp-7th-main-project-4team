# 앵커 실측 수집 갭 워크리스트

> 작성 2026-07-08. 기준선 = `data/brand/raw/브랜드별 사이즈 수집 - {긴팔,바지}.csv`(이미 수집된 실측). 이 문서는 그 위에 "빠진 것=수집타겟 / 꼬리=드롭 / 스키마정렬"을 얹은 오버레이다.
> 소유: `data/**` = 팀장. 반영/편집은 팀장 확인 후.

## 브랜드 역할 (role)

앵커(착용경험 입력) ≠ 추천 카탈로그(출력). 브랜드에 `isAnchor` 플래그를 둔다.

| role | 브랜드 | 비고 |
|---|---|---|
| `anchor` (+target) | 유니클로 · 탑텐 · 스파오 · 에잇세컨즈 · 자라 · 무신사스탠다드 | 입력이자 추천 대상 6종 (2026-07-09 H&M 제외) |
| `target` only | 노스페이스 · **H&M** (이후 추천 브랜드 확장분) | 앵커 아님. CSV/신체범위 실측 유지 = 추천 시드. 앵커 갭분석 대상 아님 |

아래 갭 분석은 **anchor 역할 7종**에만 적용.

## 축 정의

- 상의 = 여유축: `slim / regular / loose / oversize` (긴팔 기준, 소매=팔 landmark)
- 하의 = 실루엣축: `skinny / slim / straight / tapered / wide / bootcut`
- 소재(garmentType) = 핏과 직교: 상의 `shirt/tee/knit`, 하의 `denim/slacks/chino`
- 셀 = 브랜드·성별·카테고리·소재·핏, 셀당 샘플 2~3 목표
- 실측형: garment 실측 **앵커 6종**(역산) / 신체범위형 H&M = **target-only**(앵커 아님, 중간값→body prior는 추천 시)

## ❌ 드롭 (꼬리셀 — 수집 안 함)

- **데님 skinny**: 유니클로 외 전 브랜드 부재 + 트렌드 하락 → 앵커 제외
- **남성 bootcut**: 전 브랜드 여성 전용 → 남성 앵커 제외 (여성축 오픈 시 스파오 여성 부츠컷 [BEST] 741만 부활)
- CSV의 세미와이드·리얼와이드·커브드·크링클 = **wide로 정규화**(별도 셀 X)

## 🔧 스키마 정렬 (바지 CSV)

바지 CSV `fit`이 세미와이드·커브드·크링클 등 원본 혼재 → 엔진 실루엣 매칭키(engine.js silhouette)와 불일치. **`소재`(denim/slacks/chino) + `정규화실루엣`(6종) 컬럼 추가** 권장.

## 인기 floor (참고)

강=리뷰 1,000+ 또는 시그니처/명작 · 중=100~1,000 · 약=<100 또는 아울렛 단독. 자라는 SKU 인기 검증 불가 → 핏라벨 대표로만.

## 핵심 발견 (근거)

- **슬랙스가 실착 최다 앵커 소재** — 무신사 테이퍼드/와이드 슬랙스 각 리뷰 10만+. 하의 우선순위 슬랙스 ≥ 치노 ≥ 데님.
- 무신사 "550만장"은 개별 SKU API 미확인(라인 홍보문구 가능) → 근거는 리뷰수 사용.
- 실측 cm 실물은 스파오 elandmall 일부(HTML)만 확보. 나머지·자라는 브라우저 렌더/OCR 단계 필요.
