# 진단 카드 시스템 설계 — 16유형 (상체 4 × 하체 4)

> v0.1 초안 · 2026-06-29 · 검토 후 이름/규칙 수정 전제
> 핵심: **16 = 상체 4유형 × 하체 4유형**. 카테고리(상의·하의·아우터·치마·원피스)는 각자 관련 부위만 떼서 추천. 상·하의를 **둘 다** 진단하면 16 종합 카드가 완성된다.

---

## 0. 전체 구조 한눈에

```
[진단 플로우]
  ① 기본정보(성별·키·몸무게·선호핏)
  ② 상의 착용경험 → 상체 유형(U1~U4) 역산
  ③ 하의 착용경험 → 하체 유형(L1~L4) 역산
                         │
        상체 × 하체  ───►  16 종합 카드 ★(상·하의 둘 다 할 때)

[카테고리별 개별 로직 — 하나씩]
  상의   ◄ 상체 유형
  아우터 ◄ 상체 유형 (+전체 균형)
  하의   ◄ 하체 유형
  치마   ◄ 하체 유형
  원피스 ◄ 상체 + 하체 (전체 실루엣)
```

**산출 규칙(진행에 따라):**
| 진단한 것 | 나오는 것 |
| --- | --- |
| 상의만 | 상체 유형 + **상의·아우터 추천** (반쪽 카드) |
| 하의만 | 하체 유형 + **하의·치마 추천** (반쪽 카드) |
| **상의 + 하의** | **16 종합 카드** + **원피스 추천** + 전체 코디 밸런스 |

> 반쪽일 때는 "다른 쪽도 진단하면 종합 카드 완성" 유도 → 진단 완주율↑

## 1. 상체 4유형 (U)

> 축: **라인(직선↔곡선)** × **볼륨·골격**

| 코드 | 유형 | 특징 | 진단 신호(착용경험) | 스타일 방향 |
| --- | --- | --- | --- | --- |
| **U1** | 직선 탄탄 | 가슴·어깨 직선·탄탄, 상체에 살 | 가슴/어깨 끼임, 손목 보통~굵음, 상체부터 살 | 심플·V넥·세로라인, 디테일 최소 |
| **U2** | 볼륨 역삼각 | 어깨 넓고 상체 볼륨 | 어깨 끼임 강함, 가슴 끼임 | 상체 덜기, 깊은 V·다크, 어깨 강조 X |
| **U3** | 곡선 슬림 | 어깨 좁고 상체 가벼움·곡선 | 상의 핏 뜸·남음, 손목 가늚, 어깨 좁음 | 하이웨스트·프릴·소프트, 상체 포인트 |
| **U4** | 골격 프레임 | 어깨·쇄골 골격 두드러짐 | 골격 느껴짐, 핏 뜸, 손목 골격 | 오버핏·레이어드·두꺼운 소재 |

## 2. 하체 4유형 (L)

> 축: **라인(직선↔곡선)** × **볼륨·골격**

| 코드 | 유형 | 특징 | 진단 신호(착용경험) | 스타일 방향 |
| --- | --- | --- | --- | --- |
| **L1** | 직선 일자 | 힙 평평, 일자 다리 | 힙·허벅지 여유, 일자 | 스트레이트·테이퍼드, 센터프레스 |
| **L2** | 볼륨 하체 | 힙·허벅지 볼륨 | 허벅지/힙 끼임, 허리 여유 | A라인·다크·하이웨스트, 허벅지 덮기 |
| **L3** | 곡선 슬림 | 허리 가늘고 힙 곡선, 다리 슬림 | 허리 많이 여유, 힙 약간, 다리 가늚 | 하이웨스트·플레어·스커트 |
| **L4** | 골격 롱 | 다리 길고 골격·관절 두드러짐 | 기장 김, 골격, 무릎 두드러짐 | 와이드·롱·부츠컷, 골격 덮기 |

## 3. 16 종합 카드 매트릭스 (상체 × 하체)

> 셀 = 카드 이름 · 핵심 한 줄. (이름은 초안 — 바꿔도 됨)

| 상체＼하체 | **L1 직선 일자** | **L2 볼륨 하체** | **L3 곡선 슬림** | **L4 골격 롱** |
| --- | --- | --- | --- | --- |
| **U1 직선 탄탄** | ① 올 스트레이트<br>전체 직선·미니멀, 핏감이 생명 | ② 탑슬림 페어<br>상체 심플, 하체 A라인 보완 | ③ 스트레이트 웨이브<br>상직선+하곡선, 허리 강조 | ④ 스트레이트 롱<br>상체 심플, 와이드로 길이 |
| **U2 볼륨 역삼각** | ⑤ 탑헤비 밸런스<br>상체 덜고 하체 일자 균형 | ⑥ 글래머 아워글래스<br>상하 볼륨, 허리로 균형 | ⑦ 볼륨 웨이브<br>상체 덜고 하체 곡선 살림 | ⑧ 어반 역삼각<br>상체 덜고 와이드로 안정 |
| **U3 곡선 슬림** | ⑨ 소프트 스트레이트<br>상곡선+하직선, 하이웨스트 | ⑩ 러블리 페어<br>상체 포인트, 하체 A라인 | ⑪ 올 웨이브<br>전체 곡선·소프트·하이웨스트 | ⑫ 웨이브 롱<br>상곡선, 롱으로 길이 강조 |
| **U4 골격 프레임** | ⑬ 내추럴 스트레이트<br>오버 상의 + 일자 하의 | ⑭ 내추럴 페어<br>레이어 상의 + A라인 | ⑮ 내추럴 웨이브<br>골격 상체 + 곡선 하체 | ⑯ 올 내추럴<br>전체 골격·오버·롱·레이어 |

## 4. 진단 플로우 → 유형 역산 (스코어링)

각 유형에 점수를 더해 **최고점 = 확정 유형**. (착용경험 입력이 신호)

**상체(U) 스코어**
| 신호 | U1 직선 | U2 볼륨 | U3 곡선 | U4 프레임 |
| --- | --- | --- | --- | --- |
| 어깨 끼임 | +1 | **+2** | | |
| 가슴 끼임 | +1 | **+2** | | |
| 상의 핏 뜸·남음 | | | **+2** | +1 |
| 손목 가늚 | | | +2 | +1 |
| 손목 굵음·골격 | +1 | +1 | | +2 |
| 상체부터 살 | +2 | +1 | | |

**하체(L) 스코어**
| 신호 | L1 일자 | L2 볼륨 | L3 곡선 | L4 롱 |
| --- | --- | --- | --- | --- |
| 허벅지 끼임 | | **+2** | | |
| 힙 끼임 | | +2 | +1 | |
| 허리 여유 | | +1 | **+2** | |
| 힙·허벅지 여유(평평) | **+2** | | | +1 |
| 기장 늘 김 | | | | **+2** |
| 다리 골격·무릎 | +1 | | | +2 |

> ⚠️ 동점·저신뢰 시 → "옷 더 추가" 유도 (결과 카드 state와 연결). 사진 업로드(선택)는 v2 보정 입력.

## 5. 카테고리별 추천 로직 (5개 · 각자 독립)

각 로직 = `입력(유형) → 출력(잘 맞는 / 피할 / 핵심 포인트)`

### 5-1. 상의 (입력: 상체 U)
| 유형 | 잘 맞는 | 피할 |
| --- | --- | --- |
| U1 | V넥·셔츠·테일러드, 세로라인 | 프릴·퍼프·과한 디테일 |
| U2 | 깊은 V·다크·세로 분할 | 보트넥·퍼프·어깨 패드 |
| U3 | 라운드·프릴·셔링·소프트 니트 | 박시·딱딱한 재킷 |
| U4 | 오버핏·두꺼운 소재·레이어 | 얇고 몸에 붙는 핏 |

### 5-2. 아우터 (입력: 상체 U + 기장 균형)
| 유형 | 잘 맞는 | 피할 |
| --- | --- | --- |
| U1 | 테일러드 싱글 코트 | 과한 장식 |
| U2 | 다크·심플·세로, 어깨 자연스럽게 | 패드·더블 강조 |
| U3 | 숏기장·허리선 있는·소프트 | 롱·박시 |
| U4 | 오버·롱·터프한 소재 | 얇고 짧은 |

### 5-3. 하의 (입력: 하체 L)
| 유형 | 잘 맞는 | 피할 |
| --- | --- | --- |
| L1 | 스트레이트·테이퍼드, 센터프레스 | 과한 와이드 |
| L2 | 다크·하이웨스트·부츠컷, 허벅지 덮기 | 스키니·밝은 하의 |
| L3 | 하이웨스트·슬림·플레어 | 박시·로우라이즈 |
| L4 | 와이드·롱·부츠컷 | 짧은 기장·크롭 |

### 5-4. 치마 (입력: 하체 L)
| 유형 | 잘 맞는 | 피할 |
| --- | --- | --- |
| L1 | H라인·펜슬·미디 | 과한 플레어 |
| L2 | A라인·다크·무릎 덮는 길이 | 타이트 미니 |
| L3 | 플레어·하이웨스트·미디 | H라인 타이트 |
| L4 | 롱·플리츠·머메이드 | 미니·짧은 |

### 5-5. 원피스 (입력: 상체 U + 하체 L = 전체 실루엣)
> 상체 처리 + 하체 처리 결합 규칙. 예시:
| 조합 예 | 추천 |
| --- | --- |
| U1×L1 (올직선) | I라인 시프트·셔츠 원피스 |
| U2×L2 (상하 볼륨) | 허리 강조 랩 원피스 |
| U3×L3 (올곡선) | 하이웨스트·플레어 원피스 |
| U4×L4 (올프레임) | 롱 셔츠 원피스·레이어드 |
| 그 외 12조합 | "상체 규칙 + 하체 규칙" 합성으로 도출 |

## 6. 결과 카드 구성 (UI 산출물)

```
[16 종합 카드]
 ├ 상단: 카드 이름 + 상체 U·하체 L 뱃지 (예: ⑥ 글래머 아워글래스 · U2×L2)
 ├ 한 줄 정체성 + 핵심 포인트
 ├ 카테고리 탭: [상의] [하의] [아우터] [치마] [원피스]
 │     └ 각 탭 = 해당 로직의 잘맞는/피할/포인트
 └ (별개 레이어) 브랜드별 추천 사이즈 표 ← 사이즈 번역 엔진
```

> **16유형 = "어떤 핏·실루엣이 맞나"(스타일링)**, **사이즈 번역 = "브랜드별 몇 사이즈"(치수)** — 두 출력이 한 카드에 공존. 서로 독립 엔진.

## 8. 열린 질문 (검토용)

1. **상체/하체 4유형의 축**이 (라인 × 볼륨/골격)으로 맞나? 다른 프레임(골격진단 3유형 등) 쓰면 4가 안 떨어짐 → 4분류 기준 확정 필요.
2. **16 카드 이름** — 위 초안 톤(직관 설명형) vs 캐릭터형(이모지+별명) 중 어느 쪽?
3. **원피스 12조합** — 합성 규칙으로 자동 생성할지, 16개 다 수기 정의할지.
4. 진단 입력에서 **상의/하의 부위 질문**을 지금 착용경험(브랜드+핏)에 어떻게 얹을지 (옷 선택 + 부위별 느낌 확장).
